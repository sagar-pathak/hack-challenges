Welcome to the Hacking Challenge!

This assignment is centered on introducing the basic concept of vulnerability identification on websites using browser tools, such as Developer Tools. Browser Developer Tools, commonly referred to as 'DevTools,' encompass a collection of built-in utilities and features provided by web browsers. They serve the purpose of aiding developers in inspecting, debugging, and troubleshooting web pages and web applications.

In this assignment, students will be presented with one example along with its solution, followed by a similar activity that students should complete. Student should take the screenshot of each solution they have found and submit them in a single doc or pdf file in the Canvas.
